Tentang Website

Website ini dibuat menggunakan aplikasi Visual Studio Code, menggunakan domain dari Github, menggunakan framework dari Bootstrap, menggunakan library dari AOS, menggunakan API dari OpenWeatherMap, dan menggunakan 3 bahasa pemrograman yaitu HTML, CSS, dan JS.
